# 📊 データエクスポート機能 設計書

## 🎯 目的

経費データをExcel/PDF形式でエクスポートする機能を実装する。

---

## 📋 要件定義

### 機能要件

1. **経費一覧のExcelエクスポート**
   - 検索条件に基づく経費一覧をExcel形式でダウンロード
   - ヘッダー行、データ行、集計行を含む
   - セルの書式設定（通貨、日付など）
   - 最大10,000件のデータをサポート

2. **経費詳細のPDFエクスポート**
   - 個別経費の詳細情報をPDF形式でダウンロード
   - ヘッダー、詳細情報、承認履歴を含む
   - 領収書画像の埋め込み（将来対応）

3. **月次レポート（Excel/PDF）**
   - 月次集計レポートをExcel/PDF形式でダウンロード
   - 合計金額、件数、ステータス別集計
   - グラフ（円グラフ、棒グラフ）の生成（将来対応）

4. **大量データのストリーミング処理**
   - メモリ効率的な処理（100,000件以上対応）
   - ストリーミングダウンロード

### 非機能要件

1. **パフォーマンス**
   - 1,000件のデータを3秒以内にエクスポート
   - 10,000件のデータを10秒以内にエクスポート
   - メモリ使用量を最小限に抑える（ストリーミング処理）

2. **セキュリティ**
   - アクセス制御（自分の経費 or 承認者のみ）
   - ファイル名のサニタイズ

3. **拡張性**
   - 他のフォーマット（CSV、JSON）への拡張を見据えた設計

## 📊 使用ライブラリ

### Apache POI（Excel）

- **バージョン**: 5.2.3
- **機能**:
  - Excel（.xlsx）ファイルの生成
  - セルの書式設定
  - ヘッダー、フッターの設定

### iText（PDF）

- **バージョン**: 5.5.13.3（無料版）
- **機能**:
  - PDFファイルの生成
  - テーブル、段落、画像の挿入
  - フォント設定（日本語対応）

## 🔐 セキュリティ対策

### 1. アクセス制御

```java
// 経費の所有者または承認者のみエクスポート可能
if (!canAccessExpense(user, expense)) {
    throw new BusinessException("アクセス権限がありません");
}
```

### 2. ファイル名のサニタイズ

```java
// ダウンロードファイル名をサニタイズ
String filename = sanitizeFilename("経費一覧_" + date + ".xlsx");
```

### 3. データ量の制限

```java
// 一度に最大10,000件までエクスポート
if (expenses.size() > 10000) {
    throw new BusinessException("データ量が多すぎます（最大10,000件）");
}
```
